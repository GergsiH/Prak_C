# Проект: Разделяемая память и семафоры

В этом проекте я реализовала систему межпроцессного взаимодействия через разделяемую память с использованием семафоров для синхронизации. Целью было создать надёжный механизм обмена данными между независимыми процессами.

## Что я реализовала

### 1. Система разделяемой памяти с управлением доступом
Создала структуру для безопасного обмена данными между процессами через общую область памяти.

**Что делала:**
- Выделяла разделяемую память с помощью `shm_open()` и `mmap()`
- Создавала именованные семафоры для синхронизации доступа
- Реализовала структуру данных с заголовком для управления состоянием
- Обеспечивала атомарность операций чтения/записи

### 2. Модель "Производитель-Потребитель" на разделяемой памяти
Реализовала классическую задачу с несколькими процессами-производителями и потребителями.

**Что делала:**
- Создала кольцевой буфер в разделяемой памяти
- Использовала два семафора для подсчёта: свободных и занятых слотов
- Добавила мьютекс для защиты критических секций
- Реализовала graceful shutdown всех процессов


## Технологии
**Язык:** C (POSIX)
**Среда:** Linux
**Системные вызовы:** `shm_open()`, `mmap()`, `sem_open()`, `fcntl()`
**Инструменты:** GCC, GDB, Valgrind, ipcs/ipcrm
