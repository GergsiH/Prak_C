# Проект: Конвертер кодировок UTF-8 ↔ UTF-16

В этом проекте я реализовала утилиту для конвертации текстовых файлов между кодировками UTF-8 и UTF-16. Целью было глубоко разобраться в устройстве Unicode, форматах кодирования и работе с многобайтовыми символами на низком уровне.

## Что я реализовала

### 1. Декодирование UTF-8
Написала парсер UTF-8, который корректно обрабатывает символы разной длины (от 1 до 4 байт), проверяет корректность UTF-8 последовательностей и обрабатывает символы за пределами Basic Multilingual Plane (BMP).

### 2. Кодирование в UTF-16
Преобразовала Unicode code points в UTF-16 с поддержкой как little-endian, так и big-endian версий, включая обработку суррогатных пар для символов U+10000 и выше и опцию добавления BOM.

### 3. Обратное преобразование UTF-16 → UTF-8
Создала полную двунаправленную конвертацию с автоматическим определением порядка байт по BOM и корректной обработкой суррогатных пар.

### 4. Командная утилита с поддержкой потоков
Разработала консольную утилиту с поддержкой стандартных потоков ввода/вывода, обработкой ошибок и работой с большими файлами через буферизацию.


## Технологии
- **Язык:** C (стандарт C11)
- **Платформа:** кросс-платформенный
- **Инструменты:** GCC/Clang, Make/CMake
