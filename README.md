# Системное программирование на языке C

Этот репозиторий содержит мои проекты по низкоуровневому программированию на языке C. Здесь я отрабатываю работу с фундаментальными структурами данных, управление памятью и взаимодействие с операционной системой.

## Проекты

### 1. Работа со словарями (Hash Tables)
Реализация хэш-таблиц с различными методами разрешения коллизий для эффективного хранения и поиска данных.

**Что делала:** создавала собственные структуры хэш-таблиц, реализовывала цепочки (chaining) и открытую адресацию, тестировала производительность при разных нагрузках, оптимизировала хэш-функции.

### 2. Матричные операции
Обработка и манипуляции с матрицами с акцентом на эффективное использование памяти и производительные вычисления.

**Что делала:** реализовывала базовые операции (сложение, умножение, транспонирование), работала с разреженными матрицами, оптимизировала кэш-локальность, сравнивала различные способы хранения матриц в памяти.

### 3. Межпроцессное взаимодействие (IPC) и семафоры
Работа с механизмами синхронизации процессов и разделяемыми ресурсами в Unix-подобных системах.

**Что делала:** создавала разделяемую память (shared memory), использовала семафоры для синхронизации, реализовывала классические задачи (производитель-потребитель, читатели-писатели), отлаживала состояния гонок (race conditions).

### 4. Собственная реализация командной оболочки (Shell)
Создание упрощённой версии командной оболочки с поддержкой базовых функций Unix-шелла.

**Что делала:** парсила командную строку, реализовывала пайпы (pipes), настраивала перенаправление ввода/вывода, запускала внешние программы через fork/exec, обрабатывала сигналы, поддерживала фоновые процессы (jobs control).

## Технические детали
- **Язык:** C (стандарт C11/C17)
- **Платформа:** Linux/Unix системы
- **Компилятор:** GCC с флагами `-Wall -Wextra -Werror`
- **Инструменты:** GDB для отладки, Valgrind для проверки утечек памяти, Make для сборки
- **Фокус:** безопасность, эффективность и чистота кода
