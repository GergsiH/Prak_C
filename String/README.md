# Проект: Сортировка текста в алфавитном порядке

В этом проекте я разработала утилиту для сортировки текстовых данных в алфавитном порядке с поддержкой различных режимов работы и обработки Unicode-символов.

## Что я реализовала

### 1. Поддержка разных видов сортировки
Реализовала несколько режимов сортировки: по строкам целиком, по словам, с учётом и без учёта регистра, с обратным порядком (Z-A). Добавила возможность сортировки по длине строк и числовому значению при обнаружении чисел в тексте.

### 2. Обработка Unicode и локализации
Настроила корректную работу с различными кодировками (UTF-8, UTF-16) и учёт правил сортировки для разных языков. Реализовала нормализацию Unicode-строк перед сравнением для корректной обработки составных символов и диакритических знаков.

### 3. Алгоритмы сортировки и оптимизация
Имплементировала несколько алгоритмов сортировки (quicksort, merge sort, timsort) с возможностью выбора оптимального для конкретного типа данных. Добавила внешнюю сортировку для работы с файлами, превышающими размер доступной оперативной памяти.

### 4. Работа с потоками и большими файлами
Создала систему буферизированного чтения/записи для обработки крупных текстовых файлов. Реализовала многопоточную сортировку с разделением данных на блоки и последующим слиянием результатов.
