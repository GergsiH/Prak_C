# Проект: Анализатор текста и построение частотного словаря

В этом проекте я разработала программу для анализа текстовых произведений и автоматического извлечения частотного словаря с различными метриками и статистикой.

## Что я реализовала

### 1. Парсинг и предобработка текста
Создала модуль для чтения текстовых файлов в различных кодировках и форматах (plain text, HTML, Markdown). Реализовала очистку текста от разметки, приведение к нижнему регистру с учётом Unicode и токенизацию с обработкой составных слов, апострофов и дефисов.

### 2. Построение частотного словаря
Разработала систему подсчёта частотности слов с использованием хэш-таблицы собственной реализации, оптимизированной для строковых ключей. Добавила фильтрацию стоп-слов на нескольких языках и поддержку пользовательских списков исключений.
