# Проект: Библиотека для работы с матрицами

В этом проекте я разработала библиотеку для работы с матрицами, реализующую основные линейно-алгебраические операции с акцентом на производительность и точность вычислений.

## Что я реализовала

### 1. Основные арифметические операции
Создала функции для сложения, вычитания и умножения матриц с поддержкой проверки совместимости размерностей. Реализовала алгоритм умножения матриц с оптимизацией через транспонирование для улучшения локальности кэша и использования SIMD-инструкций.

### 2. Вычисление определителя и обратной матрицы
Реализовала вычисление определителя через метод Гаусса с выбором ведущего элемента для повышения численной устойчивости. Добавила функцию нахождения обратной матрицы методом Гаусса-Жордана с проверкой вырожденности.

### 3. Решение систем линейных уравнений и LU-разложение
Написала функции для решения СЛАУ методом Гаусса и через LU-разложение. Реализовала алгоритм Crout для разложения матрицы на нижнюю и верхнюю треугольные матрицы с частичным выбором ведущего элемента.

### 4. Собственные значения и векторы
Добавила вычисление собственных значений через QR-алгоритм и степенной метод для нахождения наибольшего собственного значения и соответствующего собственного вектора. Реализовала метод Якоби для симметричных матриц.
